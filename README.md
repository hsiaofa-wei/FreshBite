## 项目简介
一个基于 Vue 3 + Element Plus 的现代化水果电商平台，包含以下核心功能：
- 水果分类浏览（苹果类/柑橘类/浆果类/热带水果）
- 购物车管理（增删改查/结算）
- 用户系统（登录/注册/个人中心）
- 订单管理（创建/支付/取消）
- 收藏夹功能
- 客服中心

## 技术栈
- 前端框架：Vue 3
- 构建工具：Vite        
- 路由管理：Vue Router      
- UI组件库：Element Plus            
- 状态管理：Pinia

## Vue 项目手机预览指南
1. 确保手机和电脑连接**同一 WiFi**。
2. 在电脑上启动 Vue 开发服务器。
- 使用以下命令启动开发服务器：
    ```bash
      npm run dev
      或者
      yarn dev

- 控制台会显示如下内容：
    ```bash
  ➜  Local:   http://localhost:5173/
  ➜  Network: http://192.168.xxx.xxx:5173/ 
    ```
3. 在手机浏览器中输入上述地址，即可访问 Vue 项目。
