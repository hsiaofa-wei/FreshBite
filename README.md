# FreshBite · 新鲜水果电商平台

基于 **Vue 3 + Vite + Element Plus** 构建的现代化响应式水果购物平台，提供流畅的 PC & 移动端体验，专注于新鲜水果的分类浏览与便捷购物流程。

## 项目亮点
- 专题分类浏览：苹果、柑橘、浆果、热带水果等专题页面
- 完整购物车：添加/编辑/删除/批量结算
- 用户中心：登录注册、个人资料、订单历史、收藏夹
- 订单系统：创建订单、模拟支付、取消订单
- 客服中心：在线咨询交互入口
- 移动端友好：同一 WiFi 下手机实时预览

## 技术栈
- 前端框架：Vue 3 (Composition API)
- 构建工具：Vite（极速热更新）
- 路由管理：Vue Router（路由守卫 + 懒加载）
- UI 组件库：Element Plus（现代化设计）
- 状态管理：Pinia（轻量级、可组合）
- 包管理：npm / yarn

## 手机预览指南
1. 确保手机和电脑连接**同一 WiFi**。
2. 在电脑终端启动开发服务器：
   ```bash
   npm run dev
   # 或
   yarn dev
